# Advanced Password Generator (Python)

## 📌 Project Overview

The **Advanced Password Generator** is a Python-based command-line application that generates secure and random passwords.
It allows users to customize the password by selecting different character types such as **letters, numbers, and symbols** and choosing the number of passwords to generate.

This project demonstrates the use of **functions, loops, conditional statements, and Python built-in libraries**.

---

## ⚙️ Features

* Generate random passwords
* Choose password length
* Option to include:

  * Letters (A–Z, a–z)
  * Numbers (0–9)
  * Symbols (!, @, #, etc.)
* Generate multiple passwords at once
* Password strength indicator (Weak / Medium / Strong)
* Beginner-friendly and easy to understand

---

## 🛠 Technologies Used

* Python 3
* Built-in Python modules:

  * `random`
  * `string`
