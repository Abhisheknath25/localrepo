# Password Generator (Python)

## 📌 Project Overview

This is a simple **Python Password Generator** that creates a random and secure password using letters, numbers, and special characters.
The program asks the user to enter the desired password length and then generates a strong password.

This project is useful for beginners who want to learn **Python basics like loops, random module, and string operations**.

---

## ⚙️ Features

* Generates random passwords
* Uses letters, numbers, and symbols
* User can choose password length
* Simple and beginner-friendly code

---

## 🛠 Technologies Used

* Python 3

---

## ▶️ How to Run the Program

1. Make sure **Python** is installed on your computer.

Check Python version:

```bash
python --version
```

2. Run the program:

```bash
python main.py
```
